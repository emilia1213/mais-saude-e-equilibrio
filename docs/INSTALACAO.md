# Guia de Instalação e Desenvolvimento\n\n## 🚀 Início Rápido\n\n### Pré-requisitos\n- Node.js 16+\n- npm ou yarn\n- PostgreSQL (para produção)\n\n### 1. Clonar o Repositório\n\n```bash\ngit clone https://github.com/emilia1213/mais-saude-e-equilibrio.git\ncd mais-saude-e-equilibrio\n```\n\n### 2. Instalar Dependências\n\n#### Backend\n```bash\ncd backend\nnpm install\n```\n\n#### Frontend\n```bash\ncd ../frontend\nnpm install\n```\n\n#### Mobile (opcional)\n```bash\ncd ../mobile\nnpm install\n```\n\n### 3. Configurar Variáveis de Ambiente\n\nCrie um arquivo `.env` na pasta `backend`:\n\n```\nPORT=3001\nNODE_ENV=development\nDATABASE_URL=postgresql://usuario:senha@localhost:5432/mais-saude-equilibrio\nJWT_SECRET=sua_chave_secreta_aqui\n```\n\n### 4. Executar em Desenvolvimento\n\n#### Backend\n```bash\ncd backend\nnpm run dev\n```\n\nO servidor estará disponível em `http://localhost:3001`\n\n#### Frontend\n```bash\ncd frontend\nnpm run dev\n```\n\nA aplicação estará disponível em `http://localhost:5173`\n\n---\n\n## 📚 Estrutura do Projeto\n\n```\nmais-saude-e-equilibrio/\n├── backend/\n│   ├── src/\n│   │   ├── index.ts              #
